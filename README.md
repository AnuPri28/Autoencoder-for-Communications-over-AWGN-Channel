# Autoencoder-for-Communications-over-AWGN-Channel
Autoencoder for Communications over AWGN Channel implements an end-to-end neural communication system. The transmitter and receiver are modeled as neural networks, while the channel adds Additive White Gaussian Noise (AWGN). The model learns to encode and decode messages, improving communication reliability under noisy conditions.

## Features
- End-to-end neural communication
- AWGN channel simulation
- Neural encoder and decoder
- Model training and evaluation

## Technologies
- Python
- PyTorch
- NumPy
- Matplotlib
- Google Colab

## Files
- `Autoencoder_AWGN.ipynb` – Main notebook containing the implementation.

## How to Run
1. Open `Autoencoder_AWGN.ipynb` in Google Colab.
2. Run all cells from top to bottom.
3. View the training results and performance.
